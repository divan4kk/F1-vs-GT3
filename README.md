# 🏁 Rennsport Monza Simulation 🏎️

Kleines experimentelles C#-Projekt über die Zeitunterschiede zwischen zwei Fahrzeugklassen: F1 und GT3

## 🚗 Fahrzeuge
- F1Auto: DRS, ERS, Leistung ~1050 PS
- GT3Auto: ABS, Traktionskontrolle, Leistung ~600 PS

## 💻 Ausführen
git clone https://github.com/divan4kk/F1-vs-GT3

dotnet build

dotnet run
